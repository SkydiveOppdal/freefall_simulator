⟁ FREEFALL DRIFT

2D skydive separation simulator
Parameters
Wind Profile
Negative = wind toward jumprun (downwind run effect)
Exit Order drag to reorder
Horizontal Throw Decay (τ)
Time constant for deceleration from aircraft throw to wind equilibrium. Belly = fast, freefly = slow.
Smaller = decelerates faster from throw.
At t=3τ, ~95% of throw has dissipated.
Fall Speeds (fixed)
